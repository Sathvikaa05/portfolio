# Komara Sathvika Portfolio

A one-page recruiter-ready portfolio built with React, Vite, and Tailwind CSS, based on Komara Sathvika's resume.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Customize

Update the profile details, contact links, projects, skills, education, experience, certifications, and achievements in `src/main.jsx`.
